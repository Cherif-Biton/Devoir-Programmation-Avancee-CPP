\section{Exercice 1 -- Mod\`ele DDU : Classe Livre}

\subsection{Pr\'esentation}

Le mod\`ele DDU (D\'eclaration-D\'efinition-Utilisation) consiste \`a s\'eparer
le code en trois niveaux distincts :
\begin{itemize}[leftmargin=1.5cm]
    \item \textbf{D\'eclaration} dans le fichier \texttt{Livre.h} : prototype de la classe et de ses membres.
    \item \textbf{D\'efinition} dans le fichier \texttt{Livre.cpp} : impl\'ementation des m\'ethodes.
    \item \textbf{Utilisation} dans le \texttt{main.cpp} : appel et utilisation des m\'ethodes.
\end{itemize}
