# Devoir de Programmation Avancée — Langage C++

## Description
Devoir réalisé dans le cadre du cours de **Programmation Avancée (C++)** — Licence
Statistique et Informatique (LSI), Semestre 3  
Université Nazi Boni, Bobo-Dioulasso — Année académique 2025-2026

Ce devoir couvre les concepts avancés de la POO en C++ : classes, héritage,
classes paramétrées et STL.

## Contenu
1. **Exercice 1** — Modèle Déclaration-Définition-Utilisation (DDU) : classe Livre
2. **Exercice 2** — Héritage : classes Adresse, Personne et Eleve
3. **Exercice 3** — Classe paramétrée : classe Client
4. **Exercice 4** — STL : Vector de la classe Eleve
